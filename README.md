# IntelRadar — Public Intelligence Dashboard

Real-time intelligence dashboard for global events, finance, and crisis monitoring.

## Features
- 🌍 **Onboarding** — Country selection + topic preferences
- 📰 **Intelligence Widgets** — News, charts, stats with urgency levels
- 💹 **Financial Widgets** — Stock index, gold, oil with localized data
- 🔍 **Source Filtering** — Government, global media, local media
- 🆘 **Help Section** — Emergency contacts, hospitals, shelters, embassies
- 🔗 **Share** — Twitter/X, WhatsApp, Telegram, Facebook, LinkedIn
- 🌙 **Dark Mode** — Toggle light/dark theme
- 📱 **Responsive** — Mobile-first design

## Tech Stack
Next.js 14 · TypeScript · Tailwind CSS · Framer Motion · Recharts · Lucide Icons

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Design
Apple widget aesthetic with glassmorphism, rounded corners, subtle shadows, and smooth animations.
